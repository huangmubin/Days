# Days

一个用于记录并养成日常习惯的 App。

# 项目结构

* Days - 项目工程文件夹
    * AppDelegate - AppDelegate.swift
    * Days
        * Codebase - 链接个人代码库
        * Models - 数据模型
            * Objects - 数据对象
            * SQLites - 数据库对象
        * Views - 自定义视图
        * Controller - 控制文件（界面逻辑）
            * Habit
                * List
                * Edit
                * Show
            * Log
                * List
                * Edit
        * Profile - 配置文件
            * Color.swift
            * Font.swift
            * Formatter.swift
    * Resources - 资源文件
* Design - 项目设计文件夹
